# TaskManager

TaskManager is the best task management solution, built from the ground up with HTML, CSS, and JavaScript. No external libraries: pure, custom code for maximum performance and control.

## Features
- Create, edit, and delete tasks.
- Add date and time tags.

## Installation
1. Clone the repository.
2. Open index.html in your browser.
3. Start managing your tasks!
